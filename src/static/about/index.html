What's this project about?

This is a file sharing system meant for personal use.

It's objective is to allow an easy to setup file hosting service.

Since it's not meant for commercial use, all security relies on HTTPS. Which is where Lite comes into play.

ALL FILES uploaded to the server are stored as they are. Therefore, the server owner is able to see their contents because they're not encrypted.

It's Open Source!

The source code is available on GitHub at https://github.com/Marco4413/FileShare-Lite
